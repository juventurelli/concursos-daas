Como funciona o sync automático

Você atualiza dados na Planilha Google (Controle_Liminaristas_Gravidez)

O Apps Script (sync-sheets.gs) roda automaticamente a cada 30 minutos

O script lê as abas Liminaristas e Gravidez e converte em JSON

O JSON é enviado para o GitHub como o arquivo data.json

O dashboard carrega o data.json ao abrir — dados sempre atualizados

Como adicionar um novo painel

Crie o novo dashboard HTML (igual ao de Liminaristas)

Publique em um novo repositório GitHub e ative o GitHub Pages

Abra hub-page/index.html, encontre os cards "Novo painel em breve" e substitua pelo novo painel com o link

Faça commit e push — a página principal atualiza automaticamente
